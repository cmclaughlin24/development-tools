## Getting Started

```bash
# 1) Clone the repository
$ git clone https://github.com/NZQCPC_gme/development.git

# 2) Add the execute permission to `init.sh`
$ chmod +x ~/path/to/init.sh

# 3) Execute `init.sh`
$ ~/path/to/init.sh
```

## Installed Tools

### Homebrew

[Homebrew](https://brew.sh/) open-source package manager for designed for macOS and Linux. It is used in my development workflow to manage the installation process for a variety of tools.

#### Homebrew Taps

- [isen-ng/homebrew-dotnet-sdk-versions](https://github.com/isen-ng/homebrew-dotnet-sdk-versions): Contains different versions of the .NET SDK that can be run in parallel to the latest version.
